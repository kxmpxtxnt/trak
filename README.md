# Trak

Trak (hipper version of 'track') is an opensource multiplatform app which helps (Soon™) to track food, sports, and even
your period.

Furthermore, there should be the possibility to do this tracking with your friends to push each other.

# Goals

- Create an opensource, free to use, state-of-the-art tracking app
- Only store and process required data of users
- Well documentate the whole thing (scary!!!)
- Create a tracking device (pt3000) which is easy and cheap to build yourself
- Integrate LLMs in a modern smart way to make experience better for users
